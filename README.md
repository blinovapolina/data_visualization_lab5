# dash_viz

Работа выполняется в тех же группах, что и ЛР4.

1. Возьмите два инструмента для построения дашборда: Plotly Dash и любой второй на ваш выбор.*
2. Постройте по данным из ЛР4 дашборд. Количество элементов в дашборде – от 3 до разумного количества, достаточного, чтобы передать основную мысль. Элементом называется одно число с подписью, один график или одна таблица. Дополнительные визуальные элементы, не отражающие данные содержательно (картинка, например), могут быть в любом количестве от 0 до количества, не противоречащего здравому смыслу.
